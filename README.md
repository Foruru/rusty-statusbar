# Rusty status bar
Simple status bar for dwm written in rust. Easy to use.

## What it can show
   - Time
   - Date
   - Temperature
   - Keyboard layout

### Installation
For installation just use
```
$ cd RustyStatusBar
$ make
$ sudo make install
```
and then put it into your dwm start script
```
rusty_status_bar --loop &
```
